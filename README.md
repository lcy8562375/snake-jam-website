# Time Acceleration Website

Official company website for Shenyang Time Acceleration Technology Co., Ltd.

## Purpose

This site presents the Time Acceleration studio, the featured mobile puzzle game
Snake Jam, and the public support, privacy, and terms pages needed for platform
review and player support.

## Pages

- `/` - Company homepage
- `/about` - Company information
- `/games` - Snake Jam game page
- `/support` - Player and platform support
- `/privacy` - Privacy Policy
- `/terms` - Terms of Service
- `/robots.txt` - Search crawler rules
- `/sitemap.xml` - Sitemap for indexing

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Deployment

This project is intended to be deployed through Vercel from the GitHub
repository `lcy8562375/snake-jam-website`.

Recommended production URLs:

- Website: `https://www.timeaccel.com/`
- Support URL: `https://www.timeaccel.com/support`
- Privacy Policy URL: `https://www.timeaccel.com/privacy`
- Terms URL: `https://www.timeaccel.com/terms`

## Product Facts

- Snake Jam currently has no Time Acceleration account system.
- The public support email is `contact@timeaccel.com`.
- Store links should only be added after final App Store listing URLs are
  confirmed.
